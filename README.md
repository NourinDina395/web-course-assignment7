## Project Name
KeenKeeper

--
## Short Description
KeenKeeper is a modern relationship management web app that helps users track and nurture their personal and professional connections. It allows users to monitor interaction history, set reminders, and stay consistent with communication.


--
## Technologies Used
- React.js
- Tailwind CSS
- React Router DOM
- Node.js
- Express.js
- Lucide React Icons

--
##  Key Features

# 1. Contact Management
Add and organize contacts with tags like WORK, FAMILY, HOBBY, and TRAVEL.

# 2.  Smart Reminder System
Automatically tracks last contact date and shows status like Overdue, Almost Due, and On-Track.

# 3. Interaction Timeline
Displays a clean timeline of calls, messages, video chats, and meetups to help maintain relationships effectively.
